retryOrDefault

suspend fun <R> retryOrDefault(retries: Long = 3, base: Duration = 200.milliseconds, factor: Double = 2.0, fallback: suspend (Throwable) -> R, block: suspend () -> R): R(source)

Executes a suspend operation with retry and returns a fallback value if all retries fail.

Return

The result of successful execution or the fallback value

Example usage:

val user = retryOrDefault(retries = 3, fallback = { User.anonymous() }) {
    userService.fetchUser("user123")
}
