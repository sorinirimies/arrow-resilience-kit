ParallelSagaCoordinator

class ParallelSagaCoordinator(source)

Parallel saga coordinator for executing multiple sagas concurrently.

Useful for scenarios where multiple independent sagas need to run in parallel, such as processing a batch of orders.

Example:

val coordinator = ParallelSagaCoordinator()

val results = coordinator.executeAll(
    orders.map { order ->
        saga {
            step("Process", { processOrder(order) }, { cancelOrder(it) })
        }
    }
)
